## üõ†Ô∏è Installation & Setup

### 1. Prerequisite Compilers/Runtimes
Ensure you have the following installed on your system (Arch Linux: `sudo pacman -S ...`):
* **C/C++**: `gcc`, `g++`
* **Java**: `jdk-openjdk`
* **Python**: `python`, `python-pip`
* **JavaScript**: `nodejs`, `npm`

### 2. Running Code
* **C++**: `g++ main.cpp -o main && ./main`
* **Java**: `javac Main.java && java Main`
* **Python**: `python main.py`
* **Node.js**: `node app.js`
